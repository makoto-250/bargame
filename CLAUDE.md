## プロジェクト概要

**Bar Tycoon** は、日本のアニメ風バー経営をテーマにしたバニラ JavaScript 製のアイドル/クリッカーゲームです。Web アプリを Capacitor でラップして Android/iOS のモバイルアプリとして配布します。UI テキストおよびコンテンツはすべて日本語です。

## 開発コマンド

ビルドツール・バンドラー・トランスパイラは使用していません。ゲームはブラウザで直接動作します。

**ローカル実行**: `index.html` をブラウザで開くか、任意の静的ファイルサーバーで配信:
```
npx serve .
```

**モバイルラッパーへのデプロイ**: 変更ファイルを `www/` に手動でコピーしてから同期:
```
npx cap sync android
```

**Android APK ビルド**:
```
cd android
./gradlew assembleDebug
```

テストおよびリンター設定はありません。

## アーキテクチャ

ゲーム本体は [game.js](game.js)（約1800行）、[index.html](index.html)、[style.css](style.css) の3ファイルで完結しています。`www/` ディレクトリは Capacitor ラッパー用にこれらをミラーリングしています。

### 状態管理

すべての可変ゲーム状態は [game.js](game.js) の単一の `state` オブジェクトに格納され、`barTycoon_v1` というキーで localStorage に永続化されます。音声設定は `barTycoon_sound` に別途保存されます。状態は10秒ごと、およびタブの表示切り替え時に自動保存されます。

### コアデータオブジェクト

5つの静的データオブジェクトがゲームコンテンツを定義します:

- **`BARTENDERS`** — 雇用可能な10種類のスタッフ。基本コスト・収入・所持数・コスト倍率を持つ。収入 = `baseIncome × owned × staffMultiplier × barIncomeMultiplier`。
- **`UPGRADES`** — クリック力を上げる5種類のメニューアイテム。コストは `2.2^level` でスケール。
- **`STAFF_UPGRADES`** — スタッフごとの収入倍率（×2/×3/×5）約30種。アンロックチェーン形式。
- **`BAR_UPGRADES`** — バーポイントで購入する9種類のプレステージショップアイテム。永続的なグローバル倍率を付与。
- **`KANBAN_GIRLS`** — 収入マイルストーン到達または購入でアンロックされる4人のコスメティックキャラクター。

### ゲームループ

- `requestAnimationFrame` が `tick()` を駆動し、デルタタイムで自動収入を蓄積。
- 収入 > 0 の場合のみ、描画更新を約30 FPS にキャップ。
- グラスクリックで `clickPower` 分の金額が即時加算され、DOM にフローティングテキストが生成される。

### プレステージ（転生）

累計¥100万獲得ごとに1バーポイントを獲得。転生するとお金・バーテンダー・アップグレード・スタッフアップグレードがリセットされるが、バーポイントと実績は引き継がれる。バーポイントは BAR ショップで永続倍率と交換できる。

### ランク進行

グラスのランク（1〜5）は収入/秒のしきい値（0 / 1K / 20K / 150K / 700K 円/秒）で決まる。ランク変化時に BGM 切り替え・グラス/背景画像変更・トースト通知が発生する。

### オフライン収益

ロード時に前回セーブからの経過時間を計算し、最大8時間分の自動収入をモーダルで表示する。

### 音声

5つの BGM トラックが各ランクに対応。クリック SFX（`glasstap_2.wav`）はポリフォニーのため3つのプールを使用。BGM/SFX のトグルと音量は `barTycoon_sound` に保存される。

### UI

ボトムナビゲーションで開く4つのスライドインショップパネル（バーテンダー・スタッフアップグレード・メニューアップグレード・BAR ショップ）を持つシングルページ HTML。モーダルで実績・オプション・転生確認・データリセット確認・オフライン収益を表示。最大幅480pxのモバイルファーストデザイン。
